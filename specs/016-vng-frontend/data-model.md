# Data Model: VNG Kenniscentrum Innovatie Frontend

**Feature**: 016-vng-frontend | **Date**: 2026-06-19
**Source**: spec.md Key Entities + research.md (D3–D9). Extends existing `server/src/types/graph.ts`.

This feature is **additive** to the existing graph model. No existing field changes meaning; new node/edge kinds, new request flags, a snapshot registry, and config are introduced.

---

## 1. Graph model extensions (`server/src/types/graph.ts`)

### NodeType (extend enum)

| Value | New? | Meaning | `id` scheme | `nameId` |
|-------|------|---------|-------------|----------|
| `SPACE_L0/L1/L2` | existing | Space at hierarchy level | Alkemio UUID | space nameID |
| `ORGANIZATION` | existing | Org (incl. gemeentes) | Alkemio UUID | org nameID (e.g. `gemeente-groningen`) |
| `USER` | existing | User | Alkemio UUID | — |
| **`INITIATIVE`** | **new** | GemeenteDelers initiative (a KB Callout) | Callout UUID | callout nameID (`alkemio_nameid`) |
| **`THEME`** | **new** | GemeenteDelers theme | `theme:<slug>` (synthetic, canonical) | `<slug>` |

### GraphNode — additional optional fields (only on new kinds)

```typescript
interface GraphNode {
  // … all existing fields unchanged …
  isGemeente?: boolean;        // true when an ORGANIZATION matches the municipality registry
  // INITIATIVE-only:
  initiativeYear?: number;     // from gd-<jaartal> tag, if present
  initiativeClassifications?: string[];  // winner|finalist|kanshebber|publieksprijs|global-goals-prijs
  globalGoals?: string[];      // sdg-NN tags, if present
  sourceUrl?: string | null;   // original vng.nl/praktijkvoorbeelden link (from Callout contribution)
}
```

### EdgeType (extend enum)

| Value | New? | Source → Target | Notes |
|-------|------|-----------------|-------|
| `CHILD` / `MEMBER` / `LEAD` / `ADMIN` | existing | (space hierarchy / role edges) | unchanged |
| **`INITIATIVE_GEMEENTE`** | **new** | `INITIATIVE → ORGANIZATION` | one per referenced gemeente; reuses existing org node |
| **`INITIATIVE_THEME`** | **new** | `INITIATIVE → THEME` | one per referenced theme; shared theme node |

`GraphEdge` shape is unchanged (existing `sourceId/targetId/type/weight/scopeGroup`). New edges set `scopeGroup = null` (they belong to the GD layer, not an L0 scope).

### Validation / invariants

- **No duplicate gemeente identity**: when folding an initiative, resolve its gemeente via the registry to an `alkemioNameId`, then **reuse** the `ORGANIZATION` node whose `nameId` equals it (build `Map<nameId, node>` once); only if absent, resolve+add the org node a single time (FR-040/043, SC-015).
- **Canonical themes**: a theme referenced by N initiatives yields exactly **one** `THEME` node `theme:<slug>` (FR-041).
- **Removability**: all `INITIATIVE`/`THEME` nodes and `INITIATIVE_*` edges are tagged as GD-layer so disabling the toggle removes exactly them, leaving the base graph intact (FR-042).
- **Gemeente flag**: `isGemeente` is set by matching `ORGANIZATION.nameId` against the municipality registry (D6); drives the show/hide toggle without false positives (FR-035).

## 2. Snapshot registry (`server/src/data/vng/`)

Generated by `scripts/generate-vng-snapshot.mts` from the `vng-gemeente-delers` vault. Committed. Public reference data only (no sensitive content).

### `municipalities.json`  (~342)

```jsonc
[
  { "slug": "groningen", "title": "Groningen", "alkemioNameId": "gemeente-groningen", "cbsCode": "GM0014" }
]
```

| Field | Source (vault frontmatter) | Use |
|-------|----------------------------|-----|
| `slug` | file basename of `municipalities/<slug>.md` | join key from initiative wikilinks |
| `title` | `title` | matched against Callout tag strings (gemeente names) |
| `alkemioNameId` | `alkemio_nameid` | → existing/added `ORGANIZATION` node; gemeente identification |
| `cbsCode` | `cbs_code` (optional) | reference/disambiguation |

### `themes.json`  (~92)

```jsonc
[
  { "slug": "energietransitie", "title": "Energietransitie", "priorLabels": [] }
]
```

| Field | Source | Use |
|-------|--------|-----|
| `slug` | file basename of `gemeentedelers/themes/<slug>.md` | canonical `THEME` node id (`theme:<slug>`) |
| `title` | `title` / `vng_label` | matched against Callout tag strings (theme labels) |
| `priorLabels` | `vng_prior_labels` | match historical theme labels on older callouts |

**Matching rule**: a Callout tag is normalised (trim, case-fold) and looked up against `{title} ∪ {priorLabels}` for themes and `{title}` for municipalities. Year (`gd-YYYY`), `sdg-NN`, and classification tags are recognised by pattern and stored as initiative metadata, not edges.

## 3. Selected-space set (frontend state, `frontend-vng`)

```typescript
interface SelectedSpace {
  nameId: string;
  displayName: string;
  origin: 'hub' | 'direct';     // provenance shown in the panel (FR-013)
}
interface SelectionState {
  activeHubNameId: string | null;   // default from config on first load
  hubSpaceIds: string[];            // resolved from the active hub
  directAdded: string[];            // user additions
  directRemoved: string[];          // user removals (may include hub spaces)
  showGemeentes: boolean;           // default true
  includeInitiatives: boolean;      // default false
}
// effective = (hubSpaceIds ∪ directAdded) − directRemoved, restricted to authorised spaces
```

State persists for the session (mirrors Explorer's localStorage approach). Switching hub recomputes `hubSpaceIds`; `directAdded`/`directRemoved` are retained where still applicable (FR-012, US2 scenario 3).

## 4. Config (`server/analytics.yml` → `VngConfig` in `config.ts`)

```yaml
vng:
  defaultHubNameId: ${VNG_DEFAULT_HUB_NAMEID}:           # FR-010
  gemeentedelersSpaceNameId: ${VNG_GD_SPACE_NAMEID}:gemeentedelers   # FR-045
  gdCacheTtlHours: ${VNG_GD_CACHE_TTL_HOURS}:168          # FR-046 (~1 week)
  tagCategoryMapping:                                     # FR-022
    nds:        { "1.cloud": "cloud", "2.data": "data", "3.ai": "ai", … }
    vng2030:    { "wonen en ruimte": "wonen-ruimte", "klimaat en energie": "klimaat-energie", … }
```

```typescript
interface VngConfig {
  defaultHubNameId: string | null;
  gemeentedelersSpaceNameId: string;
  gdCacheTtlHours: number;            // default 168
  tagCategoryMapping: {
    nds: Record<string, string>;      // raw space tag (lower-cased) → NDS category key
    vng2030: Record<string, string>;  // raw space tag → VNG-2030 category key
  };
}
// ServerConfig gains: vng: VngConfig
```
